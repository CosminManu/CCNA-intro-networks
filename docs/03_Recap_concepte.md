## 🔁 Recap rapide pe concepte importante

Foloseste sectiunea asta ca „cheat sheet” pe care o tot rafinezi.

### OSI (pe scurt)

* L1 Physical: semnal, cablu
* L2 Data Link: MAC, switch
* L3 Network: IP, routing
* L4 Transport: TCP/UDP (porturi)
* L5–L7: sesiune, prezentare, aplicatie

### IP Addressing

* IP = identificator unic (ex: 192.168.x.x)
* Subnet mask = delimiteaza reteaua vs host
* Gateway = „iesirea” din reteaua ta

### Subnetting (ancora)

* „imparti un array mare in bucati mai mici”
* scop: organizare + eficienta + securitate

### Switching

* switch foloseste MAC table pentru a trimite frame-ul corect
* broadcast ramane in acelasi VLAN

### VLAN

* separare logica pe acelasi switch (ca namespace)
* comunica intre VLAN-uri doar prin routing

### Routing

* router decide ruta pe baza tabelei de rutare
* static vs dinamic (ex: OSPF)

### Regula de folosire

* dupa fiecare sesiune, adaugi 1–2 randuri noi aici
* daca nu poti scrie simplu, nu e inca clar → revii pe concept
