# 🧪 Mini-exercitii standard (2–3 intrebari rapide)

Foloseste-le ca verificare dupa fiecare concept.

### OSI

1. La ce layer se ocupa IP-ul?
2. Unde intervine MAC address?
3. Care layer gestioneaza TCP/UDP?

### IP Addressing

1. Ce face subnet mask-ul?
2. Care e diferenta intre IP privat si public?
3. Ce rol are default gateway?

### Subnetting

1. De ce impartim o retea in subneturi?
2. Ce se intampla daca ai prea multe hosturi intr-un subnet?
3. Cum iti dai seama cate hosturi ai disponibile?

### Switching

1. Cum stie un switch unde sa trimita un frame?
2. Ce este MAC table?
3. Ce se intampla cu broadcast-ul?

### VLAN

1. De ce folosim VLAN-uri?
2. Pot comunica doua VLAN-uri direct?
3. Ce problema rezolva VLAN-urile?

### Routing

1. Ce face un router diferit fata de un switch?
2. Ce este o tabela de rutare?
3. Ce inseamna static vs dynamic routing?

### Regula de folosire

* daca raspunzi usor → ai inteles conceptul
* daca te blochezi → revii la teorie sau intrebi
